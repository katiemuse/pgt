Build a Process


Remember the customer ideas we integrated earlier? Now let’s make sure Product Managers get real-time notifications on their mobile phone as customers submit new ideas.


Process Builder's point-and-click tools make it easy to automate tasks, like notifying product managers of a new idea.

Criteria

Criteria determine when to execute action groups.

Actions

Actions are executed when criteria are met.



Add Your Object

The first step in building a process is choosing the object. For this notification, we'll choose Customer Ideas from the Customer Insights Object.


Decide When the Process Should Trigger


We want the process to trigger (a push notification to be sent) every time a customer submits a new idea so we set that as a criteria.


Set Up a Push Notification


Finally, choose the action that our criteria trigger. In this case every time there’s a new idea, we want our Product Managers to get real-time push notifications on their phones.



Using the same point-and-click capabilities you can build a process for any use case in your business.

Did You Know?

You can use the “Approval Process” tool to create approval processes for particular objects/records. An approval process specifies each step of approval, including from whom to request approval and what to do at each point of the process.


The Process


Just like that, with a few clicks, you can create a whole process to notify product managers, real-time, of any ideas that customers submit.
