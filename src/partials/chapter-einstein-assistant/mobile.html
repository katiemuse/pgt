Now let's see how we can make intelligent updates with voice commands, powered by Einstein, in the Salesforce mobile app.

We can use Einstein Voice Assistant to create a reminder to follow up with Mike West.

To complete this task, Einstein uses Natural Language Processing (NLP) to associate Mike’s notes and updates with specific fields within Salesforce.

Once the voice command is complete, we have the opportunity to verify that Einstein pulled the correct account information.

And we can check to see if Einstein pulled the correct contact information.

If we had any remaining tasks, Einstein would remember this context, so we wouldn’t have to input duplicate info.

After we review everything, Einstein saves the updates back to Salesforce.
