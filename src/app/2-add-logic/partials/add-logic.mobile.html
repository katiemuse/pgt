Use Lightning Components to Create an App

Lightning Components is a modern UI framework that powers Lightning App Builder. With a simple drag-and-drop, you can build responsive mobile, web, and desktop applications.

There are 3 types of Lightning Components:

  • Standard (out of the box)
  • Custom (build your own)
  • AppExchange (built by partners)

We'll use all 3 to build Egg's Product Manager app.

Standard Components List

Standard (or Base) components provide commonly used building blocks for your app. No code required.
