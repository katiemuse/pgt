Use Lightning Components to Create an App


Lightning Components is a modern UI framework that powers Lightning App Builder. With a simple drag-and-drop, you can build responsive mobile, web, and desktop applications.



There are 3 types of Lightning Components:



  • Standard (out of the box)
  • Custom (build your own)
  • AppExchange (built by partners)


We'll use all 3 to build Egg's Product Manager app.

Did You Know?

Pardot gives users the ability to use “progressive profiling” on forms. Progressive profiling allows you to collect more data over time by dynamically displaying form fields based on whether or not a user has already filled them out. For example, if a user has already submitted their job title, the next form they encounter can ask for their industry instead.

Customize Form Fields

With Pardot, form fields are completely customizable, so you can choose the fields that are most applicable to your audience.
