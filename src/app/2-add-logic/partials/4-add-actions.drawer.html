Scheduled Actions

If you'd rather an action not start immediately, create a scheduled action. Simply set the time you want it to begin, then add the action you'd like to occur.

Did You Know?

Salesforce platform comes equipped with a list of Standard Action types, but you can easily add your own, like the 'Push Notification' through the Global Actions setup page.

Did You Know?

You can use the “Approval Process” tool to create approval processes for particular objects/records. An approval process specifies each step of approval, including from whom to request approval and what to do at each point of the process.
